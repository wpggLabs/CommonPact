# Maintenance Policy

CommonPact is a limited-capacity public research project.

Maintained items include factual corrections, broken links, protocol ambiguity, schema/vector consistency, security reports, contributor guidance, and status/website alignment.

The project promises no response time, release schedule, production support, or launch.

If no maintainer remains, the repository should mark itself inactive, stop implying active implementation, preserve history, identify last verified artifacts, and invite a documented stewardship proposal.

Restart requires named maintainers, reviewed scope, current security and dependency assessment, corrected status, and a clear relationship to existing forks and profiles.
