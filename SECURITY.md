# Security Policy

CommonPact is a pre-implementation research RFC. It is not suitable for protecting real payments, transportation, rentals, deliveries, donations, identity credentials, or safety-critical operations.

Report suspected vulnerabilities through a private GitHub security advisory when available. Do not publish personal data, secrets, exploit material against real users, or unverified accusations.

In-scope research includes canonicalization ambiguity, signature bypass, signer-set confusion, stale acceptance, replay, lifecycle bypass, privacy leaks, critical-extension downgrade, and examples that could be mistaken for production guidance.

CommonPact cannot provide incident response for independent deployments. Operators remain responsible for their security programs.
