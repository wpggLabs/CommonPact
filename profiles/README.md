# CommonPact Profiles

Profiles add domain meaning and safeguards to the CommonPact core.

| Profile | Status | Purpose |
|---|---|---|
| PactRide | standalone founder-vision RFC | source domain; not currently wire-compatible with CommonPact |
| PactRental | CommonPact Research RFC package | second-domain abstraction test |
| PactDelivery | concept | multi-party custody research |
| PactFund | high-risk concept | donation and financial-boundary research |

A directory in this repository is not approval, certification, or permission to operate. See `../PROFILE_STANDARD.md` and `../PROFILE_REGISTRY.md`.
