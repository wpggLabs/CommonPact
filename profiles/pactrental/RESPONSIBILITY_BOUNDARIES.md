# PactRental Responsibility Boundaries

| Layer | Responsibilities |
|---|---|
| CommonPact core | envelope, authorization, causality, extensions, claim/receipt distinction |
| PactRental profile | rental roles, terms, lifecycle, handoff, return, evidence, privacy rules |
| Client | understandable UI, key safety, matching, disclosure, accessibility, export/import |
| Operator/community | support, moderation, incident handling, local eligibility, provider selection |
| Payment/escrow provider | custody, settlement, refunds, chargebacks, regulatory duties |
| Insurer/inspector | policy or condition claims under its own terms |
| Owner/renter | truthful statements, lawful authority, physical conduct, agreed obligations |
| Authorities/courts | legal rights, enforcement, prohibited activity, binding adjudication |

No CommonPact signature transfers legal title or guarantees asset fitness.
