# PactRide Mapping

PactRide is the source domain and first complete founder-vision RFC. It remains independently specified at https://github.com/wpggLabs/PactRide.

CommonPact does not retroactively redefine PactRide v0.1. The mapping is explanatory and supports abstraction review. A future adapter or PactRide major version would require explicit RFCs in both repositories.

See `MAPPING.md` and `../../CORE_PROFILE_MAPPING.md`.
